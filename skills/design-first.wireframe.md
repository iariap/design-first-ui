# Skill: design-first.wireframe

## Goal

Translate the structure into a low-fidelity layout.

## Inputs

- `03_structure.md`
- Global system constraints from `docs/process.md`

## Output

Produce `04_wireframe.md` with:

- Section layout
- Component placement
- Spatial relationships
- No visual styling

## Constraints

- No colors, typography, or tokens.
- Layout only. Do not change structure.
- Keep it clear enough to map to the canvas system.
