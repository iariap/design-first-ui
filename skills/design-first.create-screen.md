# Skill: design-first.create-screen

## Goal

Create a new screen using the design-first process.

## Steps

1. Update the navigation map (02a).
2. Create 03_structure.md.
3. Create 04_wireframe.md.
4. Create 05_canvas_applied.md.
5. Create 06_ui_mock.md.
6. Implement only after 06 is done.

## Constraints

- Do not skip steps.
- Do not implement UI before 06_ui_mock.md exists.
