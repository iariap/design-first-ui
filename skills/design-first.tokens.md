# Skill: design-first.tokens

## Goal

Define the global token system derived from the canvas constraints.

## Inputs

- `01_canvas_system.md`

## Output

Produce `02_ui_tokens.md` with:

- Spacing scale
- Stroke and border
- Radius
- Typography
- Color
- Layout
- Animation
- Z-index

## Constraints

- Tokens must be derived from the base unit.
- No arbitrary values.
- Tokens must be reusable across components.
