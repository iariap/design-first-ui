# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-01-31

- Initial release of the design-first UI bundle
- Global system artifacts (canvas, tokens, navigation map)
- Screen workflow rules and change classification
- Mermaid navigation map guidance (WCAG-aware colors)
- SVG UI mock rendering command and refresh rules
- Example LLM interaction flow
- Default install directory set to `.agent`

## [0.0.2] - 2026-01-31

- Require navigation map to include every navigable use case

## [0.0.3] - 2026-01-31

- Make navigation map fully markdown-only
- Remove navigation map render command
- Align installer help with `.agent` default

## [0.0.4] - 2026-01-31

- Add Mermaid label line-break guidance for navigation maps

## [0.0.5] - 2026-01-31

- Add guidance for self-loop label visibility in Mermaid maps
- Allow update without existing lock file

## [0.0.6] - 2026-01-31

- Rename lock file to `.design-first-ui.lock`
- Allow install into existing target directory

## [0.0.7] - 2026-01-31

- Add Cursor metadata headers to rules, commands, and skills
- Improve descriptions for better automatic selection
